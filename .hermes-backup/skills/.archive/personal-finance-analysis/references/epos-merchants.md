## EPOS Statement Structure

### Page Layout
- **Page 1**: Header (name, total, payment date), first batch of transactions
- **Page 2-3**: Continued transactions
- **Last page**: Point info, credit limit, revolving balance details

### Key Data Points to Extract
| Field | Where to Find |
|-------|--------------|
| Cardholder name | Near top: `Name 様` |
| Statement total | Near name or under `お支払合計金額（円）` |
| Payment date | `お支払日: YYYY年MM月DD日` |
| Statement date | `明細作成日: YYYY年MM月DD日` |
| Revolving balance | Last page: `お支払後リボ残高` or `リボ残高` |
| Revolving interest | `うち手数料` or `うち利息` (e.g. `100,000 うち手数料 4,591円`) |
| APR | Last page: `手数料率・融資利率: 18.0%` |
| Credit limit | `ご利用可能枠` |
| Available credit | `ご利用可能残額` |
| Revolving payment course | `１０万円コース` (¥100k/mo) or `５万円コース` (¥50k/mo) |

### Date Format
All dates in statement follow: `YY MM DD` (year 2000+offset, month, day)
Example: `26 06 04` = June 4, 2026
